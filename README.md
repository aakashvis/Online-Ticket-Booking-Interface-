# Online-Ticket-Booking-Interface-
Developed a premium Flight &amp; Train Ticket Booking web interface using HTML, CSS, and JavaScript.Implemented dynamic booking forms, auto city suggestions, and real-time ticket receipt generation.A premium Flight &amp; Train Ticket Booking Web App with modern UI and smooth user experience. Inspired by real-world platforms like MakeMyTrip and Goibibo.
Features
✈️ Flight & 🚆 Train booking
🎨 Glassmorphism UI design
✨ Smooth animations
🗺️ Auto city suggestions
📅 Date selection
🎫 Travel class selection
🧾 Dynamic ticket receipt
💾 localStorage support
📸 Screenshots
🏠 Booking UI

🧾 Ticket Receipt

🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
